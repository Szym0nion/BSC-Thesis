\section{Opis technologii użytych do stworzenia aplikacji mobilnej}
Aplikacja mobilna stworzona w ramach tej pracy inżynierskiej wykorzystuje zaawansowane technologie programistyczne, bazy danych i platformy wspomagające analizę czasu reakcji. Wykorzystanie wspomnianych technologii stanowi integralną część projektu, umożliwiającą nie tylko efektywne monitorowanie parametrów fizjologicznych, ale także zapewniającą skalowalność i łatwą dostępność danych dla analizy i interpretacji wyników badań. Poniżej przedstawiono kluczowe elementy technologiczne zastosowane w aplikacji:
\subsection{Android Studio i Java}
\begin{itemize}
    \item Aplikacja mobilna została zaimplementowana w środowisku Android Studio, wykorzystując język programowania Java
    \item Android Studio jest oficjalnym środowiskiem programistycznym dla platformy Android, umożliwiającym efektywną i zintegrowaną pracę nad projektami
\end{itemize}
\subsection{Firebase}
\begin{itemize}
    \item Do przechowywania i zarządzania danymi aplikacji wykorzystano Firebase, kompleksową platformę usług od Google
    \item Firebase Realtime Database została użyta do trwałego przechowywania danych dotyczących czasu reakcji, saturacji krwi i fali tętna, zapewniając synchronizację w czasie rzeczywistym
\end{itemize}
\subsection{Gradle}
\begin{itemize}
    \item System budowania projektu oparty na Gradle został użyty do efektywnego zarządzania zależnościami, konfiguracji i budowania aplikacji
    \item Gradle ułatwia integrację różnych modułów aplikacji, w tym Firebase, co przyczynia się do płynnego procesu tworzenia i aktualizacji projektu
\end{itemize}
\subsection{GitHub}
\begin{itemize}
    \item Repozytorium kodu źródłowego aplikacji zostało umieszczone na platformie GitHub
    \item GitHub umożliwia kontrolę wersji, efektywne zarządzanie kodem źródłowym, współpracę programistyczną i udostępnianie postępów pracy nad projektem
\end{itemize}
\subsection{Optymalizacja dla Precyzyjnych Pomiarów}
\begin{itemize}
    \item Podczas implementacji zwrócono uwagę na optymalizację aplikacji pod kątem precyzyjnych pomiarów, aby gwarantować dokładność wyników czasu reakcji
\end{itemize}
